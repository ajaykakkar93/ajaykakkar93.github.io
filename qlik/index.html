PostgreSQL Upgrade Note

If migrating PostgreSQL version, follow the Qlik Community guide. Note: after QPI migration to PG 12, Qlik Sense Repository Database service is replaced by postgresql-x64-12.

Select Your Upgrade Range

Select your current version and the version you want to upgrade to. The tool will show all intermediate steps.

Expert mode covers PostgreSQL migration paths for users who need to upgrade the bundled database. Feb 2021 is the pivot version — all paths run through it.

  1. Begin upgrade from current version, stepping through intermediate versions up to the last QS bundled with PostgreSQL 9.x
  2. Upgrade to Feb 2021 — take a full PostgreSQL backup before continuing
  3. Use the Qlik PostgreSQL Installer (QPI) — found in Qlik Community Downloads — to upgrade to unbundled PostgreSQL 14.x
  4. From Feb 2021 onward, upgrade to any latest version — all support PostgreSQL 14.x
Source: Qlik Community — Manual PostgreSQL Upgrade
  1. Step-upgrade to Feb 2021 using the standard upgrade path
  2. Take a full QRS backup — refer to Qlik Help Backup Guide
  3. Uninstall Feb 2021, then fresh install Feb 2021 — this upgrades PostgreSQL from 9.6 to 12.x
  4. Restore the PostgreSQL backup — refer to Qlik Help Restore Guide (PostgreSQL login required)
  5. From Feb 2021 (fresh install), upgrade to any latest version — all support PostgreSQL 12.x
