⚑ Pivot Version

Feb 2021

The fork in the road

Upgrade path

9.6.x

Fresh install

12.x

Feb 2021 is the critical pivot. Upgrading in-place stays on 9.6.x. Only a fresh install of Feb 2021 gives you PostgreSQL 12.x. This is why the Expert Mode reinstall path goes through Feb 2021 — uninstall, fresh install, restore backup, then continue upgrading.

Also includes: 7 new visualization types, improved UI, fixed user impersonation issue.

May 2021

PostgreSQL 12.5 required — but not automatic

PG 12.5

From May 2021, Qlik Sense ships with PostgreSQL 12.5. However, upgrading QS does not automatically upgrade the bundled PostgreSQL. New PG versions are only deployed on fresh install. To manually upgrade, use the Qlik PostgreSQL Installer (QPI) — this also unbundles PostgreSQL from QS.

Aug 2023+

PostgreSQL 14.x — QPI supports Feb 2022 and above

PG 14.x

Aug 2023 ships with PostgreSQL 14.x. If migrating from 9.6 or 12.5, use the Qlik PostgreSQL Installer (QPI). Note: QPI only supports QS Feb 2022 and above. Upgrade to Feb 2022 first if on an older version before running QPI.

Quick Decision Guide

Q:

On 9.6.x, want to get to 14.x?

A:

Step-upgrade to Feb 2021 → upgrade QS to Feb 2022+ → run QPI to get PG 14.x

Q:

Want 12.x without QPI?

A:

Upgrade to Feb 2021 → take QRS backup → uninstall → fresh install Feb 2021 → restore → continue upgrading

Q:

Already on 12.x, want 14.x?

A:

Ensure QS is Feb 2022 or above → run QPI from Qlik Community Downloads
