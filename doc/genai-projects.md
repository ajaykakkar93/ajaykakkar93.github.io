## Personal Gen AI Projects

### RAG Pipeline (Retrieval-Augmented Generation)
**Certified:** RAG Systems in Practice — Edureka (March 2026) — validates this work formally
**What:** Built a RAG pipeline — the core Gen AI pattern for grounding LLM responses in a custom knowledge base.
**How RAG works:** Documents → chunked → embedded → stored in vector DB → LLM retrieves relevant chunks at query time → grounded, accurate responses instead of hallucinations.
**Significance:** RAG is the foundation of most enterprise Gen AI applications (chatbots over company docs, search, knowledge Q&A). Completing a formal cert + building it = theory AND practice.
