## Custom Plugins / Extensions Built (14+)

All extensions developed and maintained as part of the Clay Logix engagement. Many are publicly available on GitHub.

| Plugin | Description |
|--------|-------------|
| **Sheet Level Access** | Restrict access to specific Qlik sheets based on user/role |
| **Variable Button** | Toggle switch that simultaneously assigns values to multiple variables |
| **Popup Chart** | Dynamically generates buttons that load different master charts based on config |
| **Trigger** | Functions as a QlikView-style trigger (object click action) |
| **Legends** | Custom legend component for dashboards |
| **Side Filter** | Side filter pane with up to 10 master objects, hidden until open button is clicked |
| **KPI Objects** | Multiple KPI types with responsive UI, supporting 1–3 measures each |
| **Sheet Navigation** | Custom navigation between sheets |
| **Export Button** | Exports chart data to Excel using Object ID |
| **Triangle Chart** | Custom triangle/delta chart visualization |
| **Qlik Sense Styler** | Styles all Qlik objects (except TEXTBOX and KPI) |
| **Date Range Picker** | Custom date range selection component |
| **Custom Filter Pane** | Advanced filter pane with default selection and two style types (List & Button) |
| **Popup Chart (Advanced)** | Enhanced dynamic chart loading |
