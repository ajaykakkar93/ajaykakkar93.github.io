## Tech Stack

- **HTML + Tailwind CSS** via CDN — no build step, GitHub Pages static hosting
- **Font Awesome 6.5** — icons
- **Google Fonts** — Inter + Space Grotesk
- **AOS** — scroll animations
- **Typed.js** — hero typewriter effect
- **Vanilla JS** — mobile nav, cert lightbox, accordion, version selector
